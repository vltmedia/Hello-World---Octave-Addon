# Hello World - Octave Addon
This is a minimal Octave Game Engine Addon, it includes a `SC_HelloWorld.oct` which contains just a simple `Hello World` text.

# Install
## Manual
Download the code and install the directory to your Octave Project's `Packages` directory.

## Addons Repo
Download it via an Addons Repo like [VLT - Octave Engine Addons](https://github.com/vltmedia/VLT---Octave-Engine-Addons)
